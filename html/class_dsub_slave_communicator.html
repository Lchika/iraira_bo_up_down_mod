bool DsubSlaveCommunicator::handle_dsub_event ( void  )

D-sub関係イベント処理関数

引数
None
戻り値
bool true:エラーなし、false:エラーあり

この関数はゴール・コース接触検知を行うため、定期的に呼ぶ必要がある
とりあえずはloop()内で実行しておけば大丈夫のはず
丁寧なつくりにするならタイマを使って定期的に実行できるようにするとよい
